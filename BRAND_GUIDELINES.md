# MES Brand Guidelines

These are the visual brand standards for MES. Apply these to all UI, marketing pages, and component work.

## Colors

| Name       | Hex       |
| ---------- | --------- |
| Main blue  | `#3533ff` |
| CTA green  | `#b1db00` |
| Light blue | `#a3e1f0` |
| Dark blue  | `#101626` |

## Typography

- **All fonts:** Inter

## Design Style

- **No rounded edges** — `border-radius: 0` on everything
- **No drop shadows** — do not use `box-shadow` or `filter: drop-shadow(...)`
- **No glow effects**
